# NextJs - Estratégias de Renderização

# Pré-renderização

## Geração estática (SSG)
  - roda no servidor
  - estático
  - gerado no momento do build
  - facilidades com SEO

## Server-side rendering (SSR)
 - roda no servidor
 - dinâmico
 - gerado em runtime
 - facilidades com SEO

## Single page application (SPA)
 - roda no cliente
 - dinâmico
 - dificuldades com SEO
